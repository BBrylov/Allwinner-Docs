# Разработка приложений для встраиваемого Linux

Эта часть охватывает знания, связанные с разработкой приложений для встраиваемого Linux. Вы можете изучать разработку приложений независимо от понимания нижележащего фреймворка драйверов встраиваемого Linux и структуры компонентов Linux. Разработка приложений включает часть отображения UI, такую как дизайн интерфейса LVGL, интерфейс Qt, часть сетевых коммуникаций, такую как коммуникация Socket, коммуникация MQTT, коммуникация HTTP и т.д. Конечно, содержание сетевой части очень обширно. Также есть коммуникация с протоколами модулей, такая как использование gpiolib для управления светодиодными индикаторами и кнопками, управление инфракрасным портом через библиотеку инфракрасных модулей, а также часть мультимедийного отображения, отображение данных с камеры на экране, обработка данных, получаемых с камеры, и потоковая передача и т.д.

На этом сайте мы в основном разделили на следующие части:

Настройка базовой среды для разработки приложений
Разработка части отображения
Сетевая разработка
Разработка части коммуникационных модулей
Аудио и видео разработка
AI разработка
