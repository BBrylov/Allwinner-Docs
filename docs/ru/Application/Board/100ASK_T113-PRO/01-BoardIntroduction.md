# Краткое описание аппаратной части Dongshan Nezha STU

* Любые вопросы по данной отладочной плате можно обсудить на нашем форуме https://forums.100ask.net/c/aw/15

## Краткое описание аппаратной части

Отладочная плата Dongshan Nezha STU - это серия отладочных плат, специально разработанных для обучения, включающая:

 * Минимальная основная плата: содержит только базовые интерфейсы для обучения и отладки, максимально выгодная по цене, **всего 149**.
 * DIY базовая плата с полным выводом всех контактов: предназначена в основном для энтузиастов-любителей DIY, позволяет самостоятельно проектировать. **Всего 29**
 * Специальная проектная базовая плата: разработана с учетом сценариев использования чипа, предназначена для обучения в сочетании с курсами. **Всего 149**
 * Модуль ядра D1s в формате штампового отверстия: для корпоративных клиентов или для разработки продуктов.

### Минимальная основная плата
Как показано на рисунке ниже, минимальная основная плата имеет:

 * Лицевая сторона: Интерфейс TYPE-C для TTL питания и отладки, подключается напрямую к USB-порту компьютера для совмещения последовательного порта отладки и питания 2-в-1, без необходимости дополнительных кабелей.
 * Лицевая сторона: Гигабитный Ethernet-интерфейс RJ45, используется в основном для сетевой загрузки системы, загрузки ядра и других операций, удобен для разработки и отладки.
 * Лицевая сторона: USB OTG интерфейс TYPE-C, используется для записи системы и работы в качестве OTG master/slave устройства.
 * Лицевая сторона: Выведен интерфейс HDMI, может использоваться для подключения монитора и других устройств.
 * Обратная сторона: Слот для TF-карты, может использоваться для отладки и загрузки системы с TF-карты.
 * Обратная сторона: Чип 256MB SPI NAND FLASH.

### DIY базовая плата с полным выводом контактов
DIY базовая плата с полным выводом контактов выводит все неиспользованные выводы минимальной основной платы на разъемы базовой платы, предоставляет всю документацию по проектированию аппаратуры, может быть самостоятельно спроектирована и произведена с помощью JLCPCB, также можно приобрести непосредственно у нас.
Предназначена в основном для любителей DIY.

### Полнофункциональная проектная базовая плата

Полнофункциональная проектная базовая плата разработана для расширения функциональности минимальной платы Nezha STU, имеет более богатый функционал, в основном используется для изучения проектов, сети, Bluetooth, аудио, дисплея, инфракрасного порта, а также модулей датчиков и других устройств.

Полнофункциональная базовая плата имеет:

* Чип модуля WIFI/Bluetooth XR829, Bluetooth поддерживает стандартный Bluetooth и Bluetooth с низким энергопотреблением, WiFi поддерживает беспроводную сетевую связь 2.4 ГГц.
* Интерфейс дисплея MIPI DSI: поддерживает максимальное разрешение 1920x1200, интерфейс совместим с референсной платой Allwinner Nezha, в дальнейшем будут доступны соответствующие модули дисплея.
* Интерфейс инфракрасного приемника IR: поддерживает прием инфракрасного сигнала.
* 3.5MM аудиовыход: поддерживает обычные 4-контактные наушники 3.5MM, может использоваться для воспроизведения музыки и записи звука.
* MIC1 MIC2: используют кремниевые микрофоны для профессиональной записи звука.
* SPEAKER: специальный интерфейс усилителя для воспроизведения звука через динамик, разъем 1.25 mm x 2 PH.
* USB TYPE-A HOST интерфейс: используется для подключения стандартных USB-устройств, таких как USB-накопители, UVC-камеры и другие устройства.
* Интерфейс PCI-E: поддерживает подключение 4G модуля, может использоваться для PPPOE связи, независимый выход в интернет.
* Разъемы: выводят все дополнительные IO, используются для расширения модулей датчиков и т.д., разъемы питания совместимы с Raspberry Pi.
