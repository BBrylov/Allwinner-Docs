# Использование Buildroot-SDK для компиляции и сборки загрузчика

* Загрузчик отладочной платы Dongshan Nezha STU состоит из 4 частей. Первая часть - это этап boot0, используется для инициализации CPU, DDR, UART, тактовых сигналов и некоторых необходимых периферийных устройств и распределения выводов, затем переход ко второй части. Вторая часть состоит из opensbi, u-boot и board.dtb, это файл boot_package.fex.
* Таким образом, общий процесс загрузки загрузчика: boot0 --> opensbi --> u-boot --> board.dtb
