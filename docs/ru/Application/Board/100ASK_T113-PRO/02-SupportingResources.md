#### Buildroot для Dongshan NezhaSTU D1-H

**Есть видео, есть документация, есть руководство**

- Эта система сборки основана на чипе Allwinner RISCV-64 Linux D1-H, адаптирована к основной версии buildroot 2022 LTS, совместима с проектными курсами и соответствующими компонентами 100ask, действительно достигнута низкая связанность и высокая доступность. Используя различные спецификации buildroot external tree, различные проекты и различные компоненты управляются отдельно, что делает их более легкими для освоения и понимания.

#### Автоматическая сборка дистрибутивов Debian/Ubuntu для Dongshan NezhaSTU

**Есть документация, есть руководство**

* Эта система основана на `debootstrap` в сочетании с версией сообщества GNU/Linux для реализации автоматической системы сборки, в основном это серия скриптов. Эта система не является компиляцией в один шаг, требуется многократное выполнение, из-за проблем с фреймворком скриптов она не очень совершенна. В настоящее время система сборки использует исходный код, предоставленный участником сообщества https://github.com/smaeul, поддержка оборудования не очень полная, но минимальная система работает без проблем.
  * https://github.com/DongshanPI/NezhaSTU-ReleaseLinux



###  Оригинальный Tina-SDK V2.0 от Allwinner

**Есть документация, есть руководство**

* Tina Linux - это встраиваемая программная система для продуктов умного оборудования, разработанная Allwinner Technology на основе ядра Linux. Tina Linux основана на пакете разработки программного обеспечения openwrt-14.07 и включает исходный код ядра Linux, драйверы, инструменты, системное промежуточное ПО и пакеты приложений, используемые при разработке системы. OpenWRT - это известный фреймворк автоматической сборки встраиваемых Linux систем с открытым исходным кодом, состоящий из скриптов Makefile и конфигурационных файлов Kconfig. Позволяет пользователям через конфигурацию menuconfig скомпилировать полную программную систему Linux, которую можно напрямую записать на машину и запустить.

  * Онлайн документация: https://d1.docs.aw-ol.com/
