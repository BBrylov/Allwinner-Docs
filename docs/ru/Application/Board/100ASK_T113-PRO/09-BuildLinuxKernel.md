# Использование Buildroot-SDK для компиляции и упаковки ядра Linux
### Отдельная компиляция различных частей





* Отдельная компиляция этапа ядра
``` shell
book@virtual-machine:~/Neza-D1/buildroot-2021$  make linux-rebuild V=1
```

* Отдельная компиляция файловой системы
  * После указания цепочки инструментов, конфигурации системы, необходимых для установки пакетов и требуемого формата, выполните следующую команду, окончательный образ будет создан в директории output/image.
``` shell
book@virtual-machine:~/Neza-D1/buildroot-2021$ make  all //полная компиляция системы
```
