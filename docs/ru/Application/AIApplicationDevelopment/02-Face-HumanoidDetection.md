# Развертывание распознавания людей и лиц на отладочной плате

## 1. Предисловие

Чип V853 оснащен ядром NPU с максимальной производительностью обработки 1 TOPS и 128KB внутреннего высокоскоростного кэша для быстрого обмена данными. Поддерживает вызовы API OpenCL, OpenVX, Android NN и ONNX, а также импорт большого количества распространенных моделей глубокого обучения. Эта глава предоставляет пример, демонстрирующий, как использовать часть NPU чипа V853 для распознавания людей и лиц.

Пакет ресурсов (содержит приложения и программное обеспечение yuvplayer): [source.zip](https://forums.100ask.net/uploads/short-url/QhzAWsxvyGjhQx0SLSOnRpoLl2.zip)

1. Директория det_demo содержит приложения и тестовые изображения
