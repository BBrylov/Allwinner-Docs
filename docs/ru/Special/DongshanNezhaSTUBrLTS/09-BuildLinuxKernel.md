# Компиляция и упаковка ядра Linux с помощью buildroot-SDK

### Отдельная компиляция отдельных компонентов

* Этап отдельной компиляции ядра

``` shell
book@virtual-machine:~/Neza-D1/buildroot-2021$  make linux-rebuild V=1
```

* Отдельная компиляция файловой системы
  * Укажите завершенную цепочку инструментов, конфигурацию системы, необходимые пакеты и требуемый формат, выполните следующую команду. Финальный образ будет находиться в директории output/images.

``` shell
book@virtual-machine:~/Neza-D1/buildroot-2021$ make  all //Полная компиляция системы
```
