# Компиляция и упаковка загрузчика с помощью buildroot-SDK

* На отладочной плате Dongshan Nezha STU загрузчик состоит из 4 частей. Первая часть - это этап boot0, который используется для инициализации CPU, DDR, UART, тактовых сигналов и других необходимых периферийных устройств и распределения выводов. После этого начинается вторая часть, состоящая из opensbi, u-boot и board.dtb в одном файле boot_package.fex.
* Таким образом, общий процесс загрузки загрузчика выглядит следующим образом: boot0 --> opensbi --> u-boot --> board.dtb
