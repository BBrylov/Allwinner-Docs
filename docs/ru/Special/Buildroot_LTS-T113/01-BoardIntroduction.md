# Краткое описание аппаратной части отладочной платы 100ASK T113 PRO

* Любые вопросы по этой отладочной плате можно обсудить на нашем форуме https://forums.100ask.net/c/aw/15



## Краткое описание аппаратной части

Отладочная плата 100ask_t113_pro - это отладочная плата с двойной архитектурой ARM, разработанная 100ask на основе D1-H/D1s, предназначенная в основном для изучения разработки проектов Linux на чипах Allwinner.

* Форма отладочной платы: основная плата + различные базовые платы
  * Полнофункциональная отладочная плата PRO, выводящая максимально все выводы IO, размер отладочной платы сохранён таким же, как и у IMX6ULL PRO. Может непосредственно закрепить наш 7-дюймовый RGB дисплей.
  * Минимальная отладочная плата T113 Lite, сохраняющая только основные интерфейсы, выводит все интерфейсы IO на базовую плату для личных проектов DIY или проверки функциональности.

### Основная плата T113-S3

![T113_S3_Core](https://cdn.staticaly.com/gh/DongshanPI/Docs-Photos@master/100ask_t113_pro/T113_S3_Core.png)

* Краткое описание параметров основного чипа: T113-S3 - это передовой процессор приложений, разработанный специально для автомобильного и промышленного рынка управления. Он интегрирует двухядерный CortexTM-A7 CPU и одноядерный HiFi4 DSP, обеспечивая эффективную вычислительную мощность. T113-S3 поддерживает декодирование всех форматов H.265, H.264, MPEG-1/2/4, JPEG, VC1 и других. Независимый аппаратный кодировщик может кодировать в JPEG или MJPEG. Интегрированные несколько ADC/DAC и аудиоинтерфейсов I2S/PCM/DMIC/OWA могут обеспечить идеальное решение для голосового взаимодействия. T113-S3 имеет богатые возможности подключения, удобные для расширения продуктов, такие как USB, SDIO, EMAC, TWI, UART, SPI, PWM, GPADC, IR TX&RX и т.д.
* Блок-схема архитектуры чипа

![T113_Bref](https://cdn.staticaly.com/gh/DongshanPI/Docs-Photos@master/100ask_t113_pro/T113_Bref.png)

* Другие конфигурации основной платы
  * Встроенная 128MB SPI NANDFLASH x1
  * Вывод всех контактов
