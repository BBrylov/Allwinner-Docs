# Компиляция и упаковка ядра Linux с использованием buildroot-SDK

### Отдельная компиляция отдельных компонентов

* Отдельная компиляция этапа ядра

``` shell
book@virtual-machine:~/Neza-D1/buildroot-2021$  make linux-rebuild V=1
```

* Отдельная компиляция файловой системы
  * Укажите готовую цепочку инструментов, конфигурацию системы, пакеты, которые требуется установить, и необходимый формат. Выполните следующую команду, итоговый образ будет находиться в директории output/image.

``` shell
book@virtual-machine:~/Neza-D1/buildroot-2021$ make  all //полная компиляция системы
```
