# Компиляция и упаковка загрузчика с использованием buildroot-SDK

* Отладочная плата Dongshan Nezha STU имеет загрузчик, состоящий из 4 частей. Первая часть — это этап boot0, который используется для инициализации ЦПУ, DDR, UART, тактовых сигналов и других необходимых периферийных устройств и распределения выводов. После этого переходит на вторую часть. Вторая часть состоит из трех компонентов: u-boot, board.dtb, образуя файл boot_package.fex.
* Общий процесс загрузки загрузчика: boot0 --> u-boot --> board.dtb
