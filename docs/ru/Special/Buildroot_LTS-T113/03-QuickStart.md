# Быстрый старт

## Включение и загрузка системы

**Внимание: по умолчанию на отладочной плате предустановлена система SPI NAND FLASH. После получения отладочной платы вам нужны только два кабеля typec для включения**

### 1. Подключение кабеля последовательного порта и питания

* Подключите два кабеля TypeC к показанным на рисунке красным прямоугольникам. Один - это порт OTG, используемый для питания и функции прошивки, другой - это кабель последовательного порта, используемый для отладки вывода. После успешного подключения вам нужно переместить переключатель питания на нижней плате в направлении порта OTG. В этот момент зеленый индикатор питания на модуле ядра также загорается, что указывает на то, что устройство может нормально работать.

* Для последовательного порта система автоматически установит драйвер устройства последовательного порта. Если он не был установлен автоматически, вы можете использовать Driver Genius для автоматической установки.

![T113-Pro_FlashSystem](https://cdn.staticaly.com/gh/DongshanPI/Docs-Photos@master/100ask_t113_pro/T113-Pro_FlashSystem.png)


* Для системы Windows

В этот момент в Диспетчере устройств Windows в разделе "Порты (COM и LPT)" появится новое устройство последовательного порта, обычно начинающееся с `USB-Enhanced-SERIAL CH9102`. Вам нужно заметить конкретный номер COM позже для последующего использования при подключении.

![QuickStart-01](https://cdn.staticaly.com/gh/DongshanPI/Docs-Photos@master/DongshanNezhaSTU/QuickStart-01.png)

На приведенном выше рисунке номер COM - это 96, поэтому номер последовательного порта, который мы будем использовать для подключения, - это 96.

* Для системы Linux

Вы можете проверить, появилось ли новое устройство /dev/tty<>. Обычно узел устройства - это /dev/ttyACM0.

![QuickStart-02](https://cdn.staticaly.com/gh/DongshanPI/Docs-Photos@master/DongshanNezhaSTU/QuickStart-02.png)

### 2. Открытие консоли последовательного порта

#### Получение инструмента последовательного порта

Используйте инструменты последовательного порта, такие как Putty или MobaXterm, для подключения к устройству отладочной платы.

* Инструмент putty можно загрузить с сайта https://www.chiark.greenend.org.uk/~sgtatham/putty/.
* MobaXterm можно загрузить с сайта https://mobaxterm.mobatek.net/ (рекомендуется).

#### Подключение к последовательному порту с использованием putty

![QuickStart-04](https://cdn.staticaly.com/gh/DongshanPI/Docs-Photos@master/DongshanNezhaSTU/QuickStart-04.png)

#### Подключение к последовательному порту с использованием MobaXterm

Откройте MobaXterm, нажмите "Session" в левом верхнем углу. В появившемся окне выберите "Serial". Как показано на рисунке, выберите номер порта (номер COM, отображаемый в диспетчере устройств COM21), скорость передачи (Speed 115200), управление потоком (Flow Control: none), и наконец нажмите "OK". Шаги показаны на рисунке ниже.

**Внимание: управление потоком (Flow Control) должно быть установлено на "none", иначе вы не сможете вводить данные в последовательный порт в MobaXterm**

![Mobaxterm_serial_set_001](https://cdn.staticaly.com/gh/DongshanPI/Docs-Photos@master/DongshanNezhaSTU/Mobaxterm_serial_set_001.png)




### 3. Вход в shell системы

После успешного открытия последовательного порта с помощью инструмента последовательного порта вы можете непосредственно нажать клавишу Enter для входа в shell. Конечно, вы также можете нажать кнопку `Reset` на плате для перезагрузки и просмотра полной информации о системе.
