## 1 Предисловие

### 1.1 Цель написания

Введение в компиляцию и упаковку U-Boot, базовую конфигурацию, использование часто используемых команд, основные методы отладки и т.д., предоставляя основу для портирования U-BOOT и разработки приложений.

### 1.2 Область применения

Этот документ применим к brandy2.0, то есть платформе U-Boot-2018.

### 1.3 Целевая аудитория

Разработчики/специалисты по сопровождению U-Boot, разработчики ядра.



## 2 Объяснение ключевых слов класса LICHEE

Пожалуйста, перейдите в директорию longan и проверьте файл .buildconfig для просмотра используемых макросов класса LICHEE.

```
LICHEE_IC    ——> Имя IC\
LICHEE_CHIP  ——> Имя платформы\
LICHEE_BOARD ——> Имя платы\
LICHEE_ARCH  ——> Архитектура\
LICHEE_BOARD_CONFIG_DIR ——> Директория уровня платы\
LICHEE_BRANDY_OUT_DIR   ——> Директория с bin-файлами\
LICHEE_PLAT_OUT         ——> Директория с временными bin-файлами платформы\
LICHEE_CHIP_CONFIG_DIR  ——> Директория IC
```
