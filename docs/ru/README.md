### 1. Разработка приложений

Этот раздел охватывает знания, связанные с разработкой приложений для встраиваемых систем Linux. Можно изучать разработку приложений отдельно, не понимая низкоуровневых драйверов встраиваемых систем Linux, фреймворков и структуры Linux. Разработка приложений включает часть отображения UI, такую как дизайн интерфейса LVGL, интерфейс QT, часть сетевой связи, такую как связь Socket, связь MQTT, связь HTTP и т.д. Конечно, сетевая часть содержит очень много контента. Также есть связь с модулями и протоколами, например, использование gpiolib для управления светодиодами и клавишами, использование библиотеки инфракрасных модулей для управления инфракрасными устройствами, а также часть мультимедийного отображения, отображение данных камеры на экране, обработка данных, полученных с камеры, а также потоковая передача и т.д.

На этом сайте мы разделили следующие части:

1.  Настройка базовой среды для разработки приложений
2. Разработка части отображения
3. Разработка сетевой части
4. Разработка части модулей связи
5. Разработка аудио и видео
6. Разработка AI
