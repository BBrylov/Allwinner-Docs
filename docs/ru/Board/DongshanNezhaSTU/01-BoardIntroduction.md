## Описание аппаратной части

Отладочная плата Dongshan Nezha STU - это серия отладочных плат, специально разработанных для образовательных целей, включающая:

 * Минимальную основную плату: содержит только самые основные интерфейсы для обучения и отладки, обеспечивая максимальную экономичность.
 * DIY-плату расширения с выведенными всеми выводами: предназначена для энтузиастов DIY, позволяет самостоятельно проектировать.
 * Специализированные платы расширения для проектов: разработаны для конкретных сценариев использования чипа, используются вместе с курсами обучения.
 * Модуль ядра D1s в корпусе с контактными площадками: предназначен для корпоративных клиентов или разработчиков продуктов.

### Минимальная основная плата
Как показано на рисунке, минимальная плата имеет следующие ресурсы:

 * Лицевая сторона: интерфейс TYPE-C для TTL-питания и отладки, при подключении к USB-порту компьютера обеспечивает одновременно отладку через последовательный порт и питание, не требуя дополнительных кабелей.
 * Лицевая сторона: разъем RJ45 гигабитного Ethernet, используется для сетевой загрузки системы, скачивания ядра и других операций, удобен для отладки и разработки.
 * Лицевая сторона: интерфейс USB OTG TYPE-C, используется для записи системы и работы в качестве OTG host/device устройства.
 * Лицевая сторона: выведен интерфейс HDMI, может использоваться для подключения монитора и других устройств.
 * Обратная сторона: разъем TF-карты, может использоваться для отладки и загрузки системы с TF-карты.
 * Обратная сторона: чип 256MB SPI NAND FLASH.

### Полнофункциональная плата для проектов

Полнофункциональная плата для проектов разработана для расширения функциональности минимальной платы Nezha STU, обладает более богатым функционалом и используется для проектного обучения: сеть, Bluetooth, аудио, дисплей, инфракрасный порт, а также модули датчиков и другие устройства.

Функции полнофункциональной платы расширения:

* Модуль чипа XR829 WIFI Bluetooth, поддерживает стандартный Bluetooth и Bluetooth Low Energy, Wifi поддерживает беспроводную сетевую связь 2.4 ГГц.
* Интерфейс дисплея MIPI DSI: поддерживает максимальное разрешение 1920x1200, интерфейс совместим с референс-платами Allwinner Nezha, будут доступны соответствующие модули дисплея.
* Интерфейс приема ИК-сигналов: поддерживает прием инфракрасных сигналов.
* Аудиовыход 3.5MM Audio OUT: поддерживает стандартные четырехконтактные наушники 3.5MM для смартфонов, может использоваться для воспроизведения музыки и записи звука.
* MIC1 MIC2: использует кремниевые конденсаторные микрофоны для профессиональной записи звука.
* SPEAKER: специальный интерфейс усилителя для воспроизведения звука через динамик, разъем 1.25 mm x2 PH.
* Интерфейс USB TYPE-A HOST: используется для подключения стандартных USB-устройств, таких как флеш-накопители, камеры с поддержкой UVC и другие устройства.
* Интерфейс PCI-E: поддерживает подключение 4G модуля, может осуществлять PPPOE-связь, независимый доступ к интернету.
* Штыревые разъемы: выведены все свободные GPIO, используются для расширения модулями датчиков и т.д., питание совместимо с интерфейсом Raspberry Pi.
