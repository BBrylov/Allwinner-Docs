#### Buildoot для Dongshan NezhaSTU D1-H

**Есть видео, есть документация, есть справочник**

- Эта система сборки основана на чипе Allwinner RISCV-64 Linux D1-H, адаптирована под основную версию buildroot 2022lts, совместима с проектными курсами и соответствующими компонентами 100ask, действительно обеспечивает низкую связанность и высокую применимость. #### Автоматическая сборка дистрибутивов Debian/Ubuntu для Dongshan Nezha STU

**Есть документация, есть справочник**

* Эта система основана на `debootstrap` в сочетании с версией сообщества GNU/Linux и реализует автоматическую систему сборки, в основном состоящую из серии скриптов и т.д. Эта система не предусматривает компиляцию всего за один шаг, требуется выполнение нескольких раз из-за проблем с фреймворком скриптов, поэтому она не совсем идеальна. В настоящее время система сборки использует исходный код, предоставленный участником сообщества https://github.com/smaeul, поддержка аппаратной части не совсем полная, но с минимальной системой проблем нет.
  * https://github.com/DongshanPI/NezhaSTU-ReleaseLinux



###  Оригинальный Tina-SDK V2.0 от Allwinner

**Есть документация, есть справочник**

* Tina Linux - это встраиваемая программная система для интеллектуальных аппаратных продуктов, разработанная Allwinner Technology на основе ядра Linux. Tina Linux базируется на пакете разработки openwrt-14.07 и включает исходный код ядра Linux, драйверы, инструменты, системное промежуточное ПО и пакеты приложений, используемые при разработке системы. OpenWrt - это известный фреймворк автоматической сборки встраиваемых Linux систем с открытым исходным кодом, состоящий из Makefile скриптов и конфигурационных файлов Kconfig. Это позволяет пользователям с помощью конфигурации menuconfig компилировать полную программную систему Linux, которую можно непосредственно записать на устройство и запустить.

  * Онлайн-документация: https://d1.docs.aw-ol.com/
