# Обзор аппаратных ресурсов

Отладочная плата 100ask_V853_pro - это гетерогенная отладочная плата ARM и RISC-V, разработанная 100ask для разработки AI-приложений, предназначенная для изучения проектов разработки на Linux для процессоров Allwinner и разработки встраиваемых AI-приложений.

Форм-фактор отладочной платы: основной модуль + различные базовые платы

Полнофункциональная отладочная плата PRO максимально выводит все выводы IO, предоставляет два интерфейса дисплея и два интерфейса камеры, четыре интерфейса Type-A, встроенные WIFI и Bluetooth.
