# Обзор программных ресурсов

Процессор V853 официально поддерживает систему Tina Linux v5.0.

Tina Linux - это встраиваемая программная система для продуктов интеллектуального аппаратного обеспечения, разработанная Allwinner Technology на основе ядра Linux.

Tina Linux v5.0 включает исходный код boot, исходный код ядра, драйверы, инструменты, системное промежуточное ПО и пакеты приложений, используемые для разработки систем Linux. Позволяет удобно настраивать, компилировать и упаковывать образы прошивки Linux.

Tina Linux v5.0 может поддерживать построение различных систем сборки openWrt и buildroot; также поддерживает отдельную компиляцию BSP (Board Support Package, пакет поддержки платы); может упаковывать пакеты прошивки, записывать их на соответствующие устройства и запускать.

(1) Поддержка множественных систем сборки

Полный SDK может поддерживать openWrt и buildroot, некоторые настроенные SDK могут сохранять только openWrt или buildroot, или даже не иметь никакой системы сборки, поддерживая только компиляцию простого BSP.

(2) Высокая степень повторного использования BSP

В Tina Linux v5.0 BSP (код boot, kernel, инструменты компиляции и упаковки и т.д.) существует независимо, поддерживает отдельную компиляцию и упаковку, быструю генерацию образа прошивки.

(3) Модификация openWrt

Tina Linux v5.0 внес множество изменений в openWrt, например, разделение компиляции ядра, корректировку директории продуктов компиляции, независимые репозитории для пакетов ПО и директорий конфигураций и т.д., что позволяет более удобно независимо собирать rootfs и сокращать SDK.



**Примечание:**

openWrt - это фреймворк автоматической сборки встраиваемых систем Linux с открытым исходным кодом, состоящий из скриптов Makefile и файлов конфигурации Kconfig. Позволяет пользователям гибко настраивать пакеты ПО через menuconfig.



**Структура директорий**

Структура директорий Tina Linux v5.0 в основном состоит из инструментов сборки, системы сборки, инструментов конфигурации, цепочки инструментов, директории конфигурации процессора, директорий ядра и boot и т.д.

Tina Linux v5.0 имеет встроенные команды быстрого перехода, позволяющие быстро переходить в соответствующие папки для редактирования и изменения.

Ниже приведено краткое описание в порядке следования директорий и команд быстрого перехода.

```
TinaLinux/
  ├── brandy                               # Хранит код boot0, uboot и т.д.
  ├── build                                # Хранит скрипты сборки системы Tina Linux
  ├── buildroot                            # Хранит файлы конфигурации buildroot и оригинальный код buildroot
  ├── build.sh -> build/top_build.sh       # Символическая ссылка на build/top_build.sh
  ├── device                               # Хранит файлы конфигурации решений для процессоров
  ├── kernel                               # Хранит код ядра разных версий
  ├── openwrt                              # Хранит оригинальный код openWrt, пакеты ПО и директории решений для процессоров
  ├── out                                  # Хранит временные файлы компиляции и финальные образы
  ├── platform                             # Хранит исходный код некоторых пакетов ПО
  ├── prebuilt                             # Хранит некоторые предварительно скомпилированные инструменты
  └── tools                                # Хранит некоторые инструменты для хоста, инструменты загрузки и упаковки
```
