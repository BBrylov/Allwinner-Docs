# Команды управления файлами
## Команда ls
> Можно использовать ls --help для просмотра официального описания. Используется для просмотра файлов, директорий и их подробных параметров в текущем каталоге.

``` bash
ls
```
Просмотр всех файлов и папок в текущем каталоге (не включая скрытые файлы).
``` bash
ls -l
```
Просмотр подробной информации о всех файлах и папках в текущем каталоге (не включая скрытые файлы).
``` bash
ls -a
```
Просмотр всех файлов и папок в текущем каталоге (включая скрытые файлы).
``` bash
ls DIR
```
Просмотр всех файлов в целевом каталоге DIR (не включая скрытые файлы). Можно добавлять дополнительные параметры ls, например `ls Desktop -la` покажет все файлы, папки, скрытые файлы и подробную информацию о правах доступа в каталоге Desktop.

## Команда cd:
> Можно использовать cd --help для просмотра официального описания. Изменяет рабочий каталог оболочки.

``` bash
cd DIR
```

Переход (переключение) в целевой каталог DIR. Например, используя `cd Desktop` можно перейти в каталог Desktop.

``` bash
cd -
```

Переключение на предыдущий каталог, в котором была выполнена команда cd DIR. Например, сначала находимся в каталоге `~`, затем используем `cd Desktop` для перехода в каталог Desktop, после чего используем `cd -` для возврата обратно в каталог `~`.

``` bash
cd .        //переход в текущий каталог
cd ..       //переход в родительский каталог
cd  ../..   //переход на два уровня вверх
```



## Команда cp:

>  Можно использовать cp --help для просмотра официального описания. Копирует указанный <исходный файл> в <целевой файл>, или копирует несколько <исходных файлов> в <целевой каталог>.

``` bash
cp -r DIR1 DIR2   //копирование каталога DIR1 в каталог DIR2
cp file1 file2	  //копирование файла file1 в file2
cp -d link DIR    //копирование символической ссылки в каталог DIR
cp -p file1 file3 //копирование файла file1 со всеми атрибутами в file3
cp -f 				//принудительное копирование файла или папки
cp -v 				//отображение подробной информации о копировании
```



## Команда mv

> Можно использовать mv --help для просмотра официального описания. Переименовывает <исходный файл> в <целевой файл>, или перемещает исходный файл в указанный <каталог>.

``` bash
mv file1 file2  //переименование file1 в file2
mv dir1 dir2	//переименование dir1 в dir2
mv file1 dir2	//перемещение file1 в каталог dir2
mv dir2 dir1/	//перемещение каталога dir2 в каталог dir1
mv -f 			//принудительное перемещение или переименование
mv -v 			//отображение подробной информации о перемещении или переименовании
```



## Команда pwd

Вывод полного пути текущего рабочего каталога.

``` bash
pwd				//отображение полного пути к каталогу
```



## Команда mkdir

Создание одного или нескольких несуществующих каталогов, или создание каталога внутри существующего каталога.

``` bash
mkdir dir1  	//создание каталога dir1
mkdir dir1 dir2 //создание каталогов dir1 и dir2
mkdir -p dir1/dir1.1  //создание каталога dir1.1 внутри каталога dir1
```

rmdir - удаление одного или нескольких каталогов, при условии что каталог не содержит файлов.

## chown

> Команда для установки владельца файла и связанной группы.

```bash
[ubuntu@book:~]$  chown root:root  test.txt  //изменение группы и ID владельца этого файла
[ubuntu@book:~]$  chown  -R root:root ./*  //изменение группы и ID владельца всех файлов и папок в этом каталоге
```



## touch

> Используется для изменения временных атрибутов файла или каталога, или для создания нового файла.

```bash
[ubuntu@book:~]$ touch file1.txt  //изменение времени для этого файла, если файл не существует, создается новый файл
```



## file

> Используется для отображения типа файла и подробной информации.

```bash
[ubuntu@book:~]$ file file.txt  	//отображение подробной информации о типе файла
[ubuntu@book:~]$ file -i file.txt  	//отображение MIME типа файла
```
## Команда chmod





```bash
[ubuntu@book:~]$  chmod ugo+x test.sh  //добавление прав на выполнение для всех пользователей
[ubuntu@book:~]$  chmod a+x test.sh  //добавление прав на выполнение для всех пользователей
[ubuntu@book:~]$  chmod a-x test.sh  //удаление прав на выполнение для всех пользователей
[ubuntu@book:~]$  chmod 755 test.sh   //установка прав для владельца rwx, для группы r-x, для остальных r-x
[ubuntu@book:~]$  chmod -R 755 ./*  //установка прав rwx r-x r-x для всех файлов и каталогов в этом пути
```



## Команда ln

> Команда для создания ссылок на файлы

```bash
[ubuntu@book:~]$  ln -s file1.txt  doc.txt  //создание символической ссылки на file1.txt с именем doc.txt
[ubuntu@book:~]$  ln file1.txt  doc.txt  //создание жесткой ссылки на file1.txt с именем doc.txt
```



## Команда tar

> Сжатие и распаковка файлов.

```bash
[ubuntu@book:~]$ tar -czf file.tar.gz file/   	//сжатие папки file/ в формат gz, архив с именем file.tar.gz
[ubuntu@book:~]$ tar -xzf file.tar.gz 	 		//распаковка архива file.tar.gz
[ubuntu@book:~]$ tar -cjf file.tar.bz  file/ 	//сжатие папки file/ в формат bz, архив с именем file.tar.bz
[ubuntu@book:~]$ tar -xjf file.tar.gz 			//распаковка архива file.tar.gz
[ubuntu@book:~]$ tar czf - file/ | split -d -b100m - file.tar.gz //разделение папки file на части размером 100M, создание архива в формате file.tar.gz
[ubuntu@book:~]$ cat file.tar.gz0* | tar zx   //распаковка многотомного архива
```



## Команда 7z

> Распаковка или сжатие архивов с расширением .7z.

```bash
[ubuntu@book:~]$ sudo apt install p7zip-full  //установка инструмента сжатия/распаковки 7z
[ubuntu@book:~]$ 7z a -t7z -r test1.7z  dir1/  //сжатие каталога dir1 в архив test1.7z
[ubuntu@book:~]$ 7z x  test1.7z -r -odir2/		//распаковка test1.7z рекурсивно в каталог dir2
```



## Команда zip

> Сжатие и распаковка архивов с расширением .zip.

```bash
[ubuntu@book:~]$ zip -r test1.zip dir1 file1 	//сжатие dir1 и file1 в архив test1.zip
[ubuntu@book:~]$ unzip -v test1.zip 	//просмотр содержимого архива test1.zip
[ubuntu@book:~]$ unzip -t test1.zip		//проверка целостности архива test1.zip
[ubuntu@book:~]$ unzip -d tmp  test1.zip //распаковка test1.zip в каталог tmp
[ubuntu@book:~]$ zip  test1.zip  -d file1	//удаление файла file1 из архива test1.zip
[ubuntu@book:~]$ unzip -v test1.zip //просмотр содержимого архива test1.zip
```



## Команда gzip

> Сжатие архивов с расширением .gz. Может сжимать только отдельные файлы.

```bash
[ubuntu@book:~]$ gzip test1.iso  //сжатие файла test1.iso в формат gz
[ubuntu@book:~]$ gunzip test1.iso.gz //распаковка архива test1.iso.gz
```



## Команда bzip2

> Сжатие и распаковка архивов .bz2.

```bash
[ubuntu@book:~]$ bzip2 file.txt  		//сжатие файла file.txt, результат file.txt.bz2
[ubuntu@book:~]$ bunzip2 file.txt.bz2   //распаковка файла file.txt.bz2
```
