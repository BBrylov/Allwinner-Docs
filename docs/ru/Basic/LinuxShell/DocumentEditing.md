# Команды редактирования документов

## Команда cat

> Отображение содержимого файла в терминале

``` bash
[ubuntu@book:~]$ cat file	//просмотр содержимого файла file
[ubuntu@book:~]$ cat 11111 > file  //перенаправление 11111 в файл file, также можно понять как запись 11111 в file
[ubuntu@book:~]$ cat /proc/cpuinfo > cpuinfo.txt //запись информации о cpuinfo в файл cpuinfo.txt
[ubuntu@book:~]$ cat -n  file	//отображение номеров строк в файле file
[ubuntu@book:~]$ cat > helloword.txt <<EOF
>Hello
>word
>1111
>EOF
[ubuntu@book:~]$ cat helloword.txt  //часто используется при обработке в скриптах
Hello
word
1111
```



## Команда echo

> Вывод строки в терминал или отображение значения переменной оболочки

```bash
[ubuntu@book:~]$ echo "hello word!" //вывод строки hello word в терминал
[ubuntu@book:~]$ echo `date` //выполнение команды date и вывод результата
[ubuntu@book:~]$ echo -e "Hello\n Word!" //вывод строки с переводом строки после Hello
[ubuntu@book:~]$ echo -e "\033[31m Hello word \033[0m" //вывод Hello word красным цветом
```



## Команда gedit

> Текстовый редактор, аналогичный VIM

```bash
[ubuntu@book:~]$  gedit file1  //после команды gedit указывается имя файла, может быть новый или существующий файл
```



## Команда rm

> rm --help для просмотра подробного официального описания. Основное назначение - удаление указанных файлов или каталогов.

```bash
[ubuntu@book:~]$ rm file    //удаление существующего файла
[ubuntu@book:~]$ rm -r dir  //удаление каталога
[ubuntu@book:~]$ rm -v file  //отображение подробной информации в процессе удаления файла
[ubuntu@book:~]$ rm -rv dir  //отображение подробной информации в процессе удаления папки
[ubuntu@book:~]$ rm -d dir	  //удаление пустого каталога
[ubuntu@book:~]$ rm -i file  //подтверждение перед удалением, по умолчанию y означает подтверждение, другие символы означают отмену
```

## Команда grep

> grep --help для просмотра подробного официального описания. Основное назначение - поиск текстовых строк.

```bash
[ubuntu@book:~]$ grep "fpu" cpuinfo.txt //поиск ключевого слова fpu в файле cpuinfo
[ubuntu@book:~]$ grep "fpu*" cpuinfo.txt //поиск ключевого слова с подстановочным знаком fpu* в файле cpuinfo
[ubuntu@book:~]$ grep -n "fpu" cpuinfo.txt //поиск ключевого слова fpu в cpuinfo с отображением номера строки
[ubuntu@book:~]$ grep -nr "fpu" DIR //поиск ключевого слова fpu во всех файлах в каталоге dir с отображением номера строки и имени файла
[ubuntu@book:~]$ grep -nwr "fpu" DIR //поиск ключевого слова fpu целиком во всех файлах в каталоге dir с отображением номера строки и имени файла
[ubuntu@book:~]$ grep -wrh "fpu" DIR //поиск ключевого слова fpu целиком во всех файлах в каталоге dir с отображением номера строки без имени файла
```

* Расширение: также можно использовать команду egrep для поиска.



## Команда more

> Отображение содержимого документа в терминале постранично.

```bash
[ubuntu@book:~]$ more file.txt   //просмотр содержимого file.txt постранично
```

* ctrl + F : прокрутка вниз на один экран.

* пробел : прокрутка вниз на один экран.

* ctrl + B : прокрутка вверх на один экран.

* q : выход.
